self.check_group_matching( grouped_objects, match_attributes )

checks weather all groups in a grouping (generated by self.group) have have corresponding objects

  • i.e. for each object in a group, there must be a object in each other group that has equal values for match_attributes
  • see Reports::UtilTest for an example
  • raises exception if no matching
self.create_tmp_file(tmp_file_name) => string

returns the path of a file in directory ENV[‘TMP_DIR’], beginning with tmp_file_name, that does not exist yet

self.group( objects, equal_attributes ) => array

groups attributes the into groups with equal values for equal_attributes, returns an array with sub-arrays for each group

  • see Reports::UtilTest for an example
